使い方

このページは、数学の学習内容を 定義 → 定理 → 証明 → 例 の流れで整理するための土台です。

ポイント
  • 証明は <details> を使って開閉しています。
  • 見出しを増やせば、分野ごとのノートにできます。
  • 数式は \( ... \) や \[ ... \] で書けます。

サンプル 1：実数の絶対値

定義

実数 \(x\) に対して、その絶対値 \(|x|\) は

\[ |x| = \begin{cases} x & (x \ge 0) \\ -x & (x < 0) \end{cases} \]

で定義される。

定理

任意の実数 \(x\) について、\(|x| \ge 0\) が成り立つ。

証明を開く / 閉じる

定義より、\(x \ge 0\) なら \(|x| = x\) であるから \(|x| \ge 0\)。

一方、\(x < 0\) なら \(|x| = -x\) である。負の数に \(-1\) をかけると正になるので、 \(-x > 0\) したがって \(|x| \ge 0\)。

以上より、任意の実数 \(x\) に対して \(|x| \ge 0\) が成り立つ。

例
  • \(|3| = 3\)
  • \(|-5| = 5\)
  • \(|0| = 0\)

サンプル 2：数列の極限

定義

数列 \(a_n\) が実数 \(L\) に収束するとは、 任意の \(\varepsilon > 0\) に対し、ある自然数 \(N\) が存在して、 \(n \ge N\) なら

\[ |a_n - L| < \varepsilon \]

が成り立つことをいう。

定理

数列 \(a_n = \frac{1}{n}\) は \(0\) に収束する。

証明を開く / 閉じる

任意の \(\varepsilon > 0\) をとる。 アルキメデス性により、\(\frac{1}{N} < \varepsilon\) となる自然数 \(N\) をとれる。

このとき \(n \ge N\) なら

\[ \left| \frac{1}{n} - 0 \right| = \frac{1}{n} \le \frac{1}{N} < \varepsilon \]

となる。したがって定義より \(\frac{1}{n} \to 0\)。

学習メモ

こういう \(\varepsilon\)-\(N\) 証明は、定義をそのまま使う練習としてかなり重要。

</div>
